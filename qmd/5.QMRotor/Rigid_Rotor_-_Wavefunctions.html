Here is the Schrödinger equation for the rigid rotor model, a model for diatomic molecules that allows the molecule to rotate with no potential energy. It’s rigid, so it does not allow the molecule to vibrate, and its center of mass is fixed at a point in space. With that model for the rotating diatomic molecule, we’ve been able to write down Schrödinger’s equation. Our next task is to figure out which wave functions, denoted as \(\psi\), solve this differential equation.

We are looking for functions \(\psi(\theta, \phi)\) that depend on the two angular coordinates, allowing the orientation of a molecule to change, which will solve this equation. We can explore Schrödinger’s equation a little bit more by using some trial and error to find the first few solutions of this wave function. As a start, let’s say, just by trial and error, that the function needs to depend on some angles. Let’s try \(\cos(\theta)\) as a possible candidate solution and see how it works.

We need to figure out if taking the derivative of that function with respect to \(\theta\), multiplying by \(\sin(\theta)\), taking the derivative again, dividing by \(\sin(\theta)\), and adding that to the second derivative with respect to \(\phi\), divided by \(\sin^2(\theta)\), multiplied by some constants, gives us back the same function we started with. So, that’s our task to figure out right now.

Let’s do that step by step. The function is simple enough; it only involves \(\theta\) and does not involve any \(\phi\). Therefore, the first derivative with respect to \(\phi\), taken once or twice, gives us zero, so that term disappears completely. The term \(\frac{d\psi}{d\theta}\) does not go away, so I need to take the derivative of \(\cos(\theta)\) with respect to \(\theta\). This gives me \(-\sin(\theta)\).

What the equation tells me to do next is to multiply that by \(\sin(\theta)\). When I multiply \(-\sin(\theta)\) by \(\sin(\theta)\), I get \(-\sin^2(\theta)\). If I then take the derivative with respect to \(\theta\) of that \(-\sin^2(\theta)\), I get \(-2\sin(\theta)\). Then I need to use the chain rule; the derivative of \(\sin(\theta)\) itself is \(\cos(\theta)\). So, the derivative of \(-\sin^2(\theta)\) is \(-2\sin(\theta)\cos(\theta)\).

Now, including everything back through this \(\theta\) derivative, what’s left is to divide it by \(\sin(\theta)\). Thus, \(\frac{-2\sin(\theta)\cos(\theta)}{\sin(\theta)}\) cancels the \(\sin(\theta)\), and I’ve got \(-2\cos(\theta)\).

Now I’m done with taking the derivatives, and in Schrödinger’s equation, we want to know if this is true for each candidate function that we stick into it. Is it true that \(-\frac{h^2}{8\pi^2\mu r^2} \times (-2\cos(\theta))\) equals a constant, which we call the energy, multiplied by our candidate wave function \(\cos(\theta)\)?

Is it true that constants times \(\cos(\theta)\) equals a different constant times \(\cos(\theta)\)? Sure, as long as that constant \(E\) is equal to \(\frac{2h^2}{8\pi^2\mu r^2}\) after canceling these negative signs. What we’ve just determined is two things: first, this candidate wave function does indeed solve the rigid rotor Schrödinger equation. When I plug this function into the Schrödinger equation, it obeys the equation. Second, we’ve found that the energy of this wave function is \(\frac{2h^2}{8\pi^2\mu r^2}\).

Verifying a solution once we have a good guess that works is not that difficult. Not every guess we make will be a valid solution to Schrödinger’s equation, of course. For example, if \(\cos(\theta)\) works, we can ask ourselves if \(\sin(\theta)\) will also solve Schrödinger’s equation. Instead of writing everything down the same way I did before, we can take the terms one by one.

If this is \(\psi\), then the derivative with respect to \(\phi\) again disappears because there are no \(\phi\) terms in there. If I take the derivative with respect to \(\theta\), the derivative of \(\sin(\theta)\) with respect to \(\theta\) is \(\cos(\theta)\). Now I’m working on this term. If I multiply that by \(\sin(\theta)\), I have \(\cos(\theta)\sin(\theta)\).

If I now take the \(\theta\) derivative of that, I need to use the product rule. The derivative of \(\cos(\theta)\) is \(-\sin(\theta)\), so \(-\sin(\theta)\sin(\theta)\) gives \(-\sin^2(\theta)\). Using the other piece, if I leave \(\cos(\theta)\) alone and take the derivative of \(\sin(\theta)\), which is \(\cos(\theta)\), when multiplied by this \(\cos(\theta)\) gives me \(\cos^2(\theta)\).

So, I have \(-\sin^2(\theta) + \cos^2(\theta)\). After that \(\theta\) derivative, I need to divide the whole thing by \(\sin(\theta)\). Dividing all that by \(\sin(\theta)\) does not simplify as cleanly as it did the first time around. \(\frac{\sin^2(\theta)}{\sin(\theta)}\) divides just fine, but the second term is going to look like \(\frac{\cos^2(\theta)}{\sin(\theta)}\).

This does not cancel cleanly. There are some trigonometric identities we could use; we could rewrite \(\cos^2(\theta)\) as \(1 - \sin^2(\theta)\), but still, this is not going to simplify down to just constants multiplying my original wave function \(\sin(\theta)\).

What that means is, even without proceeding all the way to asking ourselves if the left side equals the right side, we can see that the derivatives we’ve taken on the left side leave us with something that’s not just the original function; it’s the original function plus some extra stuff that isn’t going to end up canceling. Therefore, this candidate wave function does not solve the Schrödinger equation.

Thus, \(\psi = \cos(\theta)\) solves the Schrödinger equation, while \(\psi = \sin(\theta)\) does not. As I said, we can use trial and error. We have two trial cases: one of them worked, and one of them didn’t. Luckily, we don’t have to proceed purely by trial and error. This particular differential equation, long before the Schrödinger equation was developed, had been studied by mathematicians. The solutions to this equation are well understood, even before anyone knew that it was useful in quantum mechanical problems for the rigid rotor.

The collection of solutions includes a whole bunch of solutions to this equation. One of them we’ve seen is \(\cos(\theta)\), and in fact, I can stick any constant I want in front of that \(\cos(\theta)\). If \(n\cos(\theta)\) solves the Schrödinger equation, then \(n\cos(\theta)\) will solve it as well because each of these terms will just have an \(n\) in front of it. If I have an \(n\) on the left side, I will also have an \(n\) on the right side, and the \(n\) doesn’t affect the solutions at all.

You won’t be surprised to hear or to remember that this \(n\) is a normalization constant that we can solve for the value of, and we’ll talk about that on a different day. In addition to this solution \(\cos(\theta)\), which we’ve validated solves Schrödinger’s equation, there are some other solutions as well. I’ve left room above that because there’s an even simpler expression that also solves Schrödinger’s equation for the rigid rotor, which is just the constant \(n\), a normalization constant all by itself.

I’ll skip the math of showing you that that’s true, but these derivatives are not terribly difficult when I’m taking the derivative of a constant, so you can verify that for yourself. Just the wave function equal to a constant will solve Schrödinger’s equation. A constant times \(\cos(\theta)\) will solve Schrödinger’s equation. We’ve seen that \(n\sin(\theta)\) does not solve Schrödinger’s equation, but if I add in some \(\phi\) dependence, so \(n\sin(\theta)e^{i\phi}\), that will solve Schrödinger’s equation, as will \(n\sin(\theta)e^{-i\phi}\).

Those functions, as well as many more, will have occasion to talk about the rest of these wave functions coming up soon. There is a long list of functions that solve this Schrödinger equation or this differential equation, and we’ll explore those solutions in a little bit of detail soon. The one last thing I’ll point out about these solutions for right now is that in order to get \(\sin(\theta)\) to solve the rigid rotor Schrödinger equation, I had to stick \(e^{i\phi}\) on the end of it. The \(i\) in this expression is the imaginary constant, the square root of negative one.

So, these last two solutions I’ve written down are actually complex solutions; they’re not purely real-valued. They involve the square root of negative one in them, which seems a little bit confusing and has some important consequences that we’ll talk about in the next video lecture.
