The third postulate of quantum mechanics tells us that the allowed values of any observable property are the eigenvalues of the operator associated with that property. However, this does not quite answer the question of what value we are actually going to see when we make a measurement. This question is a little difficult to answer because quantum mechanics is inherently probabilistic; it is not deterministic.

For example, if we consider the one-dimensional particle in a box, where the wave function, let’s say in particular wave function number one, is given by

[ (x) = () n = 1, ]

the wave function itself looks like a sine wave. The probability of finding the particle at any particular position is given by \(\psi^2\), which is the wave function times its complex conjugate. Therefore, the square of this wave function is

[ ^2 = ^2(). ]

Graphically, if I plot the probability or \(\psi^2\), the graph of \(\sin^2\) looks like this: it is zero when \(x = 0\) and also zero when \(x = a\), reaching a maximum at the midpoint when \(x = \frac{a}{2}\).

Since we know that probabilities are related to the square of the wave function, all we know about the particle is the probability of finding it at different locations. For a particle in the \(\psi_1\) wave function, which is the ground state wave function for the particle in a box, it is most likely to be found in the middle of the box. The probability is highest at the center, but there is some non-zero probability of finding it at other locations as well. The only places where we will not find it, where the probability is zero, are at the edges of the box or outside of it.

Thus, all we know about the particle is its probability of being found at various locations. I cannot definitively say that it will be found at \(\frac{a}{2}\), \(\frac{a}{4}\), or any specific value; it has some probability of being measured at any point within the box. This is what I mean when I say quantum mechanics is probabilistic. Often, all we know is the probability of finding a particle at certain locations or with certain properties.

However, that does not mean we do not know anything about the value we will measure when we measure a particular property. This brings us to the fourth postulate of quantum mechanics, which states that if I have a system with a particular wave function, and I want to know the value of a specific property, I can predict something called the expectation value. This is essentially another term for the mean or average.

If I have some property \(A\) that I am interested in—perhaps that property is energy, position, momentum, or something else—we know that this property has some operator associated with it. To find the average value I will obtain after making multiple measurements, we express the expectation value as

[ A = ^* , dx, ]

where \(\hat{A}\) is the operator acting on the wave function \(\psi\), and \(\psi^*\) is the complex conjugate of the wave function. If I integrate this expression over the entire space, the result will give me a number, which will be the average value of the property I expect when I make multiple measurements.

For example, if I ask about the average position of the particle, it will be in the middle of the box. Sometimes it will be on the left side, sometimes on the right side, but on average, I expect to find it in the middle. Any single measurement will be unpredictable, but if I make a number of measurements one after the other, I can predict what the average of those measurements will be.

To calculate an expectation value, I need to know what property I am interested in, what the operator of that property is, and which wave function the system is occupying. If I want to find the average value for the position, I can use the fourth postulate to calculate that. We need to calculate the wave function times the operator associated with the position. Remember that the position operator is simply multiplication by \(x\).

It gives us information about the probability of where I will find the particle, but it does not directly indicate the momentum without additional work.

Thus, we discover that the average momentum of the particle in a box, when it is in the \(\psi_1\) wave function, is zero. This is true because a particle in this one-dimensional box, described by the \(\psi_1\) wave function, will move to the left as often as it moves to the right. Consequently, the average momentum, which is the sum of positive and negative values, cancels out to zero.

These are two examples of how to use the expectation value and the fourth postulate to calculate the expectation value for a property. We have seen that we can predict the average value of a property even when we cannot necessarily predict the specific value that a property will have. I cannot tell you exactly where the particle is, but I can tell you that on average, if I make many measurements, I will find it in the middle of the box.

However, there is one exception to my claim that quantum mechanics is probabilistic and not deterministic. There is one case where we can tell exactly what the result of a measurement will be, which we can illustrate with this example. Let’s consider one more property: the energy.

For the same particle in a one-dimensional box with wave functions \(\psi_n\), I want to know what the average energy is. Again, using the postulate, the average energy will be given by

[ E = ^* , dx, ]

where \(\hat{H}\) is the Hamiltonian operator associated with energy. When I perform this integral, I do not even have to worry about whether this is a one-dimensional particle in a box or which particular \(n\) value it is.

I know that if this is the wave function, then

[ _n = E_n _n, ]

where \(E_n\) is the energy associated with that wave function. Since \(E_n\) is just a constant, I can pull it out of the integral, and what remains is simply the square of the wave function with one of the two being the complex conjugate.

If the wave function is normalized, that integral equals 1. Therefore, the average value for the energy of a particle in a wave function \(\psi_n\) is simply

[ E = E_n. ]

This means that the only value the energy can take is the energy corresponding to that wave function. This is true not just for energy but for any property. If I have a wave function that is an eigenfunction of the operator I am inquiring about, then the only possible measured value of that property is the corresponding eigenvalue.

Since the wave functions for the one-dimensional particle in a box are not eigenfunctions of the position operator, we have many different values, and all we can predict is the average. However, for energy, the wave functions are eigenfunctions of the Hamiltonian, so they will always have that particular value.

This illustrates how to apply the fourth postulate, and we have one more postulate to cover, which we will discuss in the next video.
